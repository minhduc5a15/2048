# 2048 Web

A React + TypeScript + TailwindCSS port of the 2048 game engine.

## Features

- Core logic ported from optimized C++ bitboard implementation.
- Smooth animations using Framer Motion (Slide, Merge Pop, Spawn Zoom).
- LocalStorage persistence for Game State and High Score.
- Keyboard support (Arrow Keys & WASD).

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
